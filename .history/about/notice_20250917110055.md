# 使用须知

### 关于本站
本Wiki旨在为《Gooboo》玩家提供全面、准确的数据和攻略参考。所有内容均由社区玩家共同编辑和维护。

### 数据来源
*   大部分数据来源于游戏客户端解包。
*   部分策略和技巧总结自社区高玩经验。
*   所有数据仅供参考，请以游戏内实际效果为准。

### 参与贡献
我们欢迎所有热爱这款游戏的玩家参与到Wiki的建设中来！如果你发现了错误、有新的发现或想补充内容，请通过以下方式联系我们：
1.  在GitHub仓库提交 Issue 或 Pull Request。
2.  加入我们的玩家社区QQ群：123456789
